# 基本面类指数基金



#### 基本面指数

**挑选原则：从四个维度综合考察企业状况，给四个方面打分**

+ 企业的营业收入
+ 企业的现金流
+ 企业的净资产
+ 企业的分红



#### 三种常见的基本面指数



+ 基本面50：挑选沪深两市中基本面分数最高的50家企业
  + 场内/场外：**嘉实基本面50（160716）**
+ 基本面60：深圳交易所中60家
  + 场内：**建信深证60ETF（159916）**
  + 场外：**建信深证60联接（530015）**
+ 基本面120：深圳交易所120家
  + 场内：嘉实深证120ETF（159910）
  + 场外：嘉实深证120联接（070023）

| 基本面50指数                                             | 基本面60指数/基本面120指数                            |
| -------------------------------------------------------- | ----------------------------------------------------- |
| 沪深两市                                                 | 深市（两个指数相似度高，基本面120包含着基本面60指数） |
| 在上海交易所中，有大量的银行股，基本面50中纳入大量银行股 | 银行股较少，多为格力等制造业公司                      |
